What Worked Well

Experience Builder is a great Public-Facing web-solution.

  • Overall this web solution produces a nice interface, which can be heavily customized as there are many widgets available.
  • You can add 2D or 3D maps.
  • It is possible to have multiple pages to show different maps or different sections of a map.
  • It works well on desktop view as well as mobile view.
  • It does have a navigation component, which uses geocoded addresses. This is useful for getting directions from one location to another.
  • You can add images, charts, a legend, etc.

Lessons Learned

The biggest challenge with Experience Builder is that it takes more time than other web-solutions to fully understand how to create a quality web experience.

  • Although it has a navigation section, it is not great for indoor navigation.
  • There is a widget to get precise coordinates by clicking on the map, which users can copy/ paste these coordinates into the navigation section. This could be a work-around for choosing a precise location in the navigation (like a room or location within a building), without relying solely on geocoded addresses, but this is not very user-friendly.
  • After publishing the experience builder for the public, each route request uses credits.
  • You can add a Filter Widget to search for specific attributes within your layers, for example, room numbers, buildings, or fire hydrants. However, more experimentation with this is necessary.
